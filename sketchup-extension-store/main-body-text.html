VR Sketch is a tool for creating, editing and viewing your SketchUp models in virtual reality.
With one click send any SketchUp file to VR.
All the main SketchUp tools are available in VR so now you can stand inside your model and edit it with your hands.
Join a friend and work on the same project together in real time.

VR Sketch has a two-way live-link with SketchUp so the changes you make in VR are updated in SketchUp and vice versa.
This means there is no exporting, importing, downloading or uploading to the cloud; just work like you would in SketchUp.

Requirements
We support both the HTC Vive and Oculus Rift (Some Windows Mixed Reality headsets work as well).
SketchUp Make 2017, SketchUp Pro 2017 as well as SketchUp Pro 2018 are all supported.
See https://www.sketchup.com/download/all to download either version
