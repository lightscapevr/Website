1. Download most recent version using the link below

2. Install the extention

  • Open Sketchup
  • In the top menu bar click on "Window" then click on "Extention Manger"
  • In the bottom left click "Install"
  • Navigate to where you download the VR Sketch file. Select it and click "Install"
  • Done!

3. Have fun

VR Sketch is now installed, just click "Extention", "VR Sketch", "Send to VR" to start working in VR!
